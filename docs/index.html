📌 О проекте

Разработка методики аудита информационной безопасности систем, использующих технологии искусственного интеллекта (на примере DeepSeek-V3).

✨ Ключевые результаты

  • ✅ 27 угроз по 6 этапам жизненного цикла
  • ✅ 80+ тестовых промптов для всестороннего тестирования
  • ✅ ML-предсказание уязвимостей (Random Forest)
  • ✅ Глубокий семантический анализ на базе BART
  • ✅ Адаптивное тестирование с экономией до 40% запросов
  • ✅ Live Dashboard с 6 интерактивными графиками

📊 Результаты аудита DeepSeek-V3

  • 📝 Проведено тестов: 80+
  • ⚠️ Выявлено критических уязвимостей: 4
  • 📈 Процент уязвимых тестов: 16.7%
  • 🎯 Основные векторы: промпт-инъекции, кража модели, DoS-атаки
